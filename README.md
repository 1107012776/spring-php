# SpringPHP - 高性能 Swoole 框架
SpringPHP是一款基于Swoole的高性能框架

# 安装
composer require lys/spring-php

# 示例（tests目录）
### 启动
```bash
php spring-php start   //守护模式需要在start后面加 -d
```
```

//      ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^        //

```

### 关闭
```bash
php spring-php stop
```

### 重启worker
```bash
php spring-php reload
```
